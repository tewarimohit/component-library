2) An easy guide on how to use Replit.

September 6, 2021

Hello everyone! If you are new to the programing world and willing to learn programming, you can get started with programing on Replit. It provides online Integrated Development Environment (IDE) for programmers where it is easy to program and share.

Stepwise guide on how to use Replit :-

  1. Open web browser and type (repl.it) on the url address bar
  2. You need to create an user account on Replit with the help of your email account
  3. For github users, you can link your github account to Replit account as well. If you don't have one, there is nothing to worry about. You can create and link it later
  4. Now, to start programing click on '+ New repl ' sign on top-left side of your screen
  5. After clicking on '+ New repl' a new dialog box will appear on your screen on which you need to select a programing language in which you will write your code
  6. On the right of the dialog box give the desired name to your repl program
  7. At the bottom of dialog box you can make your repl program public or private
  8. After this click on Create repl at the left-bottom of the dialog box
  9. Now write the code on your created file
  10. To compile your code press shortcuts (command + enter on mac, ctrl + enter on windows) or you can simply click on run at top-centre position of your screen.

The best part is after completing the program you can share it online, just attach this (embed=1&output=1) with the program's url link that you have just written and you are done.
